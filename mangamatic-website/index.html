HOW IT WORKS


First, we take in a set of manga pages.

Next, we split up each page by panel, store these images in a folder, then return a JSON pointing to each image.

With each panel, we use different algorithms to determine the "sentient" of an image based on its iconography and text.

By combining these metrics, we can determine what types of transitions to apply to each panel.


Finally, an end user can view the animatic.
